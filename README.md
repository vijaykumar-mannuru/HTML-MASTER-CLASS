## 🎯 What You Will Learn

### 🔰 Basics

* What is HTML?
* Structure of HTML (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`)
* Tags and Elements
* Attributes

### 🧱 Core Topics

* Headings, Paragraphs
* Links & Images
* Lists (Ordered, Unordered)
* Tables
* Forms (Input, Label, Button, etc.)
* Semantic Tags

### 🎨 Advanced Concepts


* Iframes
* Meta Tags (SEO basics)
* HTML5 Features

## 💡 Example Code

```html
<!DOCTYPE html>
<html>
<head>
    <title>My First Page</title>
</head>
<body>

<h1>Welcome to HTML</h1>
<p>This is my first HTML program.</p>

<a href="https://google.com">Click Here</a>

</body>
</html>
```
